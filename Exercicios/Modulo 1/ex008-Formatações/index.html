Principais formatações:

Negrito/ Destaque

Nessa frase temos um termo em negrito usando a tag B (não semantica)

Ja nessa frase, temos um termo em destaque usando a tag STRONG (semântica)

Italico/ Enfase

Nesta frase temos um termo em italico, usando a tag I (não semântica)

Ja nessa, temos um termo em ênfase usando a tag EM

Texto marcado

Podemos tambem marcar um texto usando a tag MARK

veja que essa marcação não herda as caracteristicas anteriores

Texto grande e pequeno

Tambem é possivel criar textos grandes e textos pequenos

Texto deletado

Podemos marcar um texto como excluido para indicar que ele pode ser lido, mas não considerado

Texto insirido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Existe tambem como sublinhar um texto com a tag U (que não pe semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3

Texto subscrito

Para inserir coisas do tipo H2O
